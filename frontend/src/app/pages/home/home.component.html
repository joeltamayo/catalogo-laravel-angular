Descripción de Movie Manager

Movie Manager es una plataforma web diseñada para la gestión y organización de películas. Permite a los usuarios explorar un catálogo completo de películas, acceder a detalles específicos de cada título y administrar la colección de manera eficiente.

Características principales:

  • Catálogo interactivo: Visualización de las películas en tarjetas con imágenes, título, género y año de lanzamiento.
  • Gestión CRUD: Permite agregar, editar y eliminar registros.
  • Vista de detalles: Muestra información detallada de cada película, como sinopsis, director y año de publicación.
  • Interfaz amigable: Diseño moderno y responsivo para una experiencia de usuario fluida.

Ideal para:

  • Cinéfilos que desean organizar su colección personal.
  • Administradores de plataformas de streaming o videoclubs.
  • Creadores de contenido que desean llevar un control de sus recomendaciones de cine.
